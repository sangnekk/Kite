# Điều kiện so sánh

<EmbedFlowNode type="control_condition_compare" />

Khối `Comparison Condition` cho phép bạn tạo logic điều kiện bằng cách so sánh hai giá trị. Bạn có thể thiết lập nhiều nhánh so sánh để thực thi hành động theo kết quả.

Khối này tạo cấu trúc phân nhánh, nơi các nhánh khác nhau sẽ được chọn tùy theo kết quả so sánh.
