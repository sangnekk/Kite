# Bộ lọc sự kiện

<EmbedFlowNode type="option_event_filter" />

Khối `Event Filter` cho phép bạn lọc sự kiện dựa trên thuộc tính của chúng. Bạn có thể chỉ định các điều kiện bắt buộc để sự kiện kích hoạt flow.

Điều này hữu ích khi tạo bộ xử lý sự kiện cụ thể hơn, chỉ phản hồi với một số loại sự kiện hoặc đặc điểm nhất định.
