# Lấy người dùng Roblox

<EmbedFlowNode type="action_roblox_user_get" />

Khối `Get Roblox User` cho phép bạn lấy thông tin người dùng Roblox bằng ID hoặc username. Điều này hữu ích khi tạo integration với Roblox hoặc xây dựng tính năng dùng dữ liệu người dùng Roblox.

Bạn có thể tra cứu người dùng bằng Roblox ID hoặc username, và khối sẽ trả về thông tin tương ứng.
