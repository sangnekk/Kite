# Hỏi AI

<EmbedFlowNode type="action_ai_chat_completion" />

Khối `Ask AI` cho phép bạn tương tác với các mô hình trí tuệ nhân tạo. Bạn có thể đặt câu hỏi, nhận phản hồi cho prompt hoặc để AI thực hiện nhiều tác vụ xử lý văn bản.

Khối này hỗ trợ nhiều mô hình AI với khả năng và chi phí khác nhau. Phản hồi có thể được dùng ở các khối phía sau.
