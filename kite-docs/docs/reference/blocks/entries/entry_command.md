# Lệnh

<EmbedFlowNode type="entry_command" />

Khối `Command` là điểm bắt đầu cho slash command. Đây là nơi flow của lệnh bắt đầu khi người dùng gọi slash command.

Bạn có thể cấu hình tên và mô tả lệnh trực tiếp trong khối này. Lệnh sẽ được đăng ký tự động với Discord khi bạn deploy ứng dụng.
