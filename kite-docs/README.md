# Trang web

Trang web này được xây dựng bằng [Docusaurus](https://docusaurus.io/), một trình tạo trang web tĩnh hiện đại.

Đã được việt hóa bởi [@sang0023](http://sang0023.io.vn/).

### Cài đặt

```
$ npm install
```

### Phát triển cục bộ

```
$ npm run start
```

Lệnh này khởi chạy server phát triển cục bộ và mở cửa sổ trình duyệt. Hầu hết các thay đổi được phản ánh trực tiếp mà không cần khởi động lại server.

### Build

```
$ npm run build
```

Lệnh này tạo nội dung tĩnh vào thư mục `build` và có thể được phục vụ bằng bất kỳ dịch vụ hosting nội dung tĩnh nào.
